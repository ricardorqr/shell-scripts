# Install Maven, Git, Java and Ansible 

You can create a script to automate the installation process of Maven, Git, Java and Ansible. [Here's an example](/install-utils.sh).

1. Open a text editor on your Linux system.
2. Copy and paste the [example](/install-utils.sh) code into the text editor.
3. Save the file as `install.sh`.
4. Open the terminal on your Linux system and navigate to the directory where you saved the `install-utils.sh` file.
5. Run the following command to make the script executable:

```shell
chmod +x install-utils.sh
```

6. Run the script using the following command:

```shell
./install-utils.sh
```

7. Firstly, this will update your system's package list. Secondly, install Ansible, Git, Maven and Java. Lastly, check all versions.
